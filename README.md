# Home/X - Twitter-Inspired UI Clone

## Overview

**Home/X** is a modern, responsive social media interface inspired by Twitter. This project aims to replicate key elements of a Twitter-like feed, including a sidebar navigation, posts feed, reaction buttons, and a “What’s happening?” input section. It demonstrates a clean and interactive UI, designed to be fully responsive across different screen sizes.

---

## Features

* **Responsive Design:** Adapts seamlessly to different screen sizes, hiding or adjusting UI elements on smaller devices.
* **Sidebar Navigation:** Interactive sidebar with hover effects and icons for Home, Explore, Notifications, Messages, Profile, and more.
* **Posts Feed:** Displays user posts with avatars, images, and reaction buttons.
* **Interactive Reactions:** Hover effects and color transitions for likes, retweets, and other actions.
* **Sticky Elements:** Certain UI elements, like the sidebar and post tabs, remain visible during scrolling.
* **Custom Icons & Graphics:** Integrated Material Symbols and SVG graphics for a polished look.

---

## Technologies Used

* **HTML5** - Markup structure for content and layout.
* **Tailwind CSS** - Utility-first CSS framework for styling, layout, and responsiveness.
* **SVG Icons & Material Symbols** - For high-quality scalable icons.
* **Responsive Media Queries** - Custom CSS for different screen widths and heights.

## Installation & Usage

1. Clone the repository:

   ```bash
   git clone <https://github.com/muhammad-muzammil57/x-clone---By-Muzammil/tree/main>
   ```
2. Open `index.html` in your browser.
3. For customizations, edit `index.html` or `output.css`.
4. Tailwind utilities are already configured via `output.css`.
